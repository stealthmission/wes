Here is a little summary about my project:


Generated a fake JSON API with using faker.js, created a generate.js file which generated the json file. After a bit of searching on how to host my API I came across https://my-json-server.typicode.com/ which allowed me to pull a small json api from my github account. https://my-json-server.typicode.com/stealthmission/jsonAPI , https://github.com/stealthmission/jsonAPI So was able to pull data from the small json file that was generated earlier, which I thought was really cool to be able to do.

Used Angular material for their prebuilt UI's Able to consume the json api via service data Had trouble implementing a cart service as it interfered with my json objects and would not show my products, was not able to implement a cart function. Followed a tutorial for adding HTTP error handling

Was able to deploy the build to Nelify but if you were to navigate to the next page it would throw an error and not get the products. Could not find a solution to that issue.
