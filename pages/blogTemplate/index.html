CSS Style Guide

This is the Code Style guide for CSS

Posted by Wes Gilmour on April 17, 2022

Indentation

Use an indent with tabs, no spaces.

< div class="post-heading">
 <CSS Style Guide>

Capitalization in ID's and Names

ID and class names should be in lowerCamelCase.

.styleGuide {

HTML elements should be in lowercase.

body, div {

Write Valid CSS

Validate any CSS writen through a CSS validator and make sure no errors are present at time of submission.

Comments

Please include comments, if performance is an issue they can be removed with a CSS minifier.

Colors

Use capital letters when using hexadecimal notation. Three and Six digits are acceptable, if possible use 3 digit as to save a few bytes of download time.

/* Good */
color: #FFS;
color: #ET8365;
/* Bad */
color: #ffs;

Dimensions

Specify units in either em, px, or %g

/* Good */
height: 10px;
/* Bad */
height: 10;
